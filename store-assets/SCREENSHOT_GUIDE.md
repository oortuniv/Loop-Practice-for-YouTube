## ğŸ¨ Screenshot Preparation Checklist

### Before Taking Screenshots

- [ ] Use a real YouTube video (music tutorial recommended)
- [ ] Choose video with clear sections (intro, verse, chorus, etc.)
- [ ] Set up 4-5 realistic practice loops
- [ ] Fill in all fields (labels, tempo, time signature)
- [ ] Test in both light and dark mode
- [ ] Clear browser notifications/extensions

### During Screenshot

- [ ] Full screen browser (F11) or maximized window
- [ ] Hide browser bookmarks bar
- [ ] Hide YouTube's top navigation if possible
- [ ] Zoom to 100% (Ctrl+0)
- [ ] Capture at exactly 1280x800 or scale proportionally
- [ ] Ensure text is crisp and readable

### After Screenshot

- [ ] Crop to exact 1280x800
- [ ] Optimize file size (use PNG compression)
- [ ] Add subtle drop shadow if needed
- [ ] Verify text readability at thumbnail size
- [ ] Number files: 01-, 02-, 03-, etc.
