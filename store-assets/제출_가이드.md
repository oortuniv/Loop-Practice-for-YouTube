# Chrome 웹 스토어 제출 가이드

## 📦 필수 준비물

### 1. 아이콘 (필수)

**위치**: `store-assets/icons/`

- **icon-16.png** (16x16) - 확장 프로그램 툴바 아이콘
- **icon-48.png** (48x48) - 확장 프로그램 관리 페이지
- **icon-128.png** (128x128) - Chrome 웹 스토어 목록 & 설치 대화상자

**디자인 가이드라인**:
- PNG 형식, 투명 배경
- 간단하고 인식하기 쉬운 디자인
- 작은 크기에서도 잘 보여야 함
- 브랜드 색상과 일관성 유지

**현재 상태**: ⚠️ **작업 필요** - 다음을 표현하는 아이콘 제작 필요:
- 음표 또는 루프 심볼
- 기타/악기 실루엣
- YouTube 재생 버튼과의 통합

---

### 2. 스크린샷 (필수, 1-5개)

**위치**: `store-assets/screenshots/`

**필수 크기**:
- 1280x800 (16:10 비율) - 권장
- 또는 최소 640x400

**촬영할 내용**:
1. **메인 인터페이스** - 루프 섹션 카드와 컨트롤
2. **루프 생성** - 새 루프 생성 대화상자
3. **템포 설정** - BPM 및 박자 표시 컨트롤
4. **여러 루프** - 여러 연습 섹션 정리된 모습
5. **속도 조절** - 재생 속도 조정 기능 사용 중

**현재 상태**: ⚠️ **작업 필요** - YouTube에서 실제 스크린샷 캡처 필요

## 🚀 제출 단계

### 1. 개발자 계정 준비
- [Chrome 웹 스토어 개발자 대시보드](https://chrome.google.com/webstore/devconsole)로 이동
- $5 일회성 등록비 지불
- 이메일 주소 확인

### 2. ZIP 패키지 생성
```bash
cd dist
zip -r ../loop-practice-for-youtube.zip .
```

### 3. 스토어에 업로드
- 대시보드에서 "새 항목" 클릭
- ZIP 파일 업로드
- 스토어 리스팅 세부 정보 입력
- 필요한 모든 이미지 업로드

### 4. 스토어 리스팅 완성
- 설명 추가 (STORE_LISTING.md에서 복사)
- 스크린샷 업로드
- 카테고리 선택: "생산성"
- 개인정보 처리방침 URL 추가
- 지원 웹사이트/이메일 추가

### 5. 검토 제출
- "검토 제출" 클릭
- 일반적으로 검토에 1-3 영업일 소요
- 검토자 피드백이 있으면 대응
