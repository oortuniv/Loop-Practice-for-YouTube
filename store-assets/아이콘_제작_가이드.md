# 아이콘 제작 가이드

## 📐 필수 크기

**3개의 아이콘 파일**을 만들어야 합니다:

1. **icon-16.png** - 16x16 픽셀 (확장 프로그램 툴바, 작은 표시)
2. **icon-48.png** - 48x48 픽셀 (확장 프로그램 관리 페이지)
3. **icon-128.png** - 128x128 픽셀 (Chrome 웹 스토어, 설치 대화상자)

**형식**: 투명 배경 PNG
**색상 모드**: RGB
**비트 깊이**: 24비트 또는 32비트 (알파 채널 포함)

---

## 🎨 디자인 컨셉

### 테마: 음악 연습 + 루프 심볼

아이콘은 다음을 결합해야 합니다:
1. **음악 요소** (악기, 음표, 또는 음악 관련 심볼)
2. **루프/반복 심볼** (순환 화살표, 무한대, 또는 재생 아이콘)
3. **YouTube 통합** (선택사항: 재생 버튼 또는 YouTube 빨강)

### 색상 팔레트

**주요 색상**:
- **YouTube 빨강**: #FF0000 또는 #FF0033
- **YouTube 파랑**: #065fd4
- **진한 회색**: #212529 (대비용)
- **흰색**: #FFFFFF

**강조 색상**:
- **음악 금색**: #FFD700 (선택사항, 음악 요소용)
- **연습 녹색**: #00AA00 (선택사항, "활성" 상태용)

## 💡 디자인 컨셉 4가지

### 컨셉 1: 루프 화살표 + 음표 (추천!)
```
┌──────────────┐
│   ♪          │  음표 (검정/금색)
│      ↻       │  순환 화살표 (파랑)
│              │
└──────────────┘
```

**장점**: 명확하고 간단하며 인식하기 쉬움
**색상**: 파란 순환 화살표 위 검정 음표

---

### 컨셉 2: 기타 + 반복 심볼
```
┌──────────────┐
│   🎸         │  기타 실루엣
│    ⟲        │  이중 곡선 화살표
│              │
└──────────────┘
```

**장점**: 명확하게 음악 관련, 악기 중심
**색상**: 밝은 배경에 어두운 기타, 파란 화살표

---

### 컨셉 3: 재생 버튼 + 루프
```
┌──────────────┐
│   ▶          │  재생 삼각형 (YouTube 스타일)
│    ∞        │  무한대/루프 심볼
│              │
└──────────────┘
```

**장점**: YouTube 통합이 명확함
**색상**: 빨간 재생 버튼, 파란 무한대

---

### 컨셉 4: 미니멀 루프 링
```
┌──────────────┐
│              │
│     ⟳       │  점이 있는 단일 순환 화살표
│              │
└──────────────┘
```

**장점**: 매우 깔끔하고 작은 크기에서도 잘 보임
**색상**: 파랑 또는 파랑에서 빨강으로 그라데이션

## 🎯 크기별 디자인 사양

### 16x16 아이콘 (툴바)
- **간단하게 유지**: 최대 2색
- **높은 대비**: 밝은 배경과 어두운 배경 모두에서 보여야 함
- **최소 디테일**: 가는 선 피하기
- **심볼 중앙 배치**: 모든 면에 2px 패딩 남기기
- **가시성 테스트**: 다른 브라우저 테마에서 확인

**권장사항**:
```
YouTube 파랑의 간단한 순환 화살표 (⟳)
또는
검정색 음표 (♪)
```

### 48x48 아이콘 (관리 페이지)
- **더 많은 디테일 허용**: 2-3개 요소 포함 가능
- **썸네일 크기에서 명확**: 작아도 여전히 인식 가능
- **균형잡힌 구성**: 중앙 중심 디자인
- **패딩 남기기**: 6-8px 여백

**권장사항**:
```
순환 화살표 (⟳) + 음표 (♪)
또는
기타 실루엣 + 루프 심볼
```

### 128x128 아이콘 (스토어 리스팅)
- **완전한 디테일**: 텍스트, 그라데이션, 효과 포함 가능
- **브랜드 정체성**: 가장 눈에 띄는 버전
- **전문적 품질**: 이것이 "상점 정면"
- **패딩 남기기**: 16-20px 여백

**권장사항**:
```
그라데이션 배경 (파랑에서 연한 파랑)
음표가 있는 큰 순환 화살표
선택사항: "LP" 텍스트 또는 악기 실루엣
깊이감을 위한 미묘한 그림자
```

## 🎨 단계별 디자인 과정

### Figma 사용 (초보자에게 권장)

1. **새 파일 생성** (128x128)
2. **원 그리기** (108x108, 중앙 배치)
3. **순환 화살표 경로 추가**
   - 호 도구 사용
   - 화살촉 추가
   - 색상: #065fd4 (YouTube 파랑)
4. **음표 추가**
   - 텍스트 또는 벡터 사용
   - 폰트: 볼드, sans-serif
   - 색상: #212529 (진한 회색)
5. **내보내기**:
   - 128x128 → icon-128.png
   - 48x48로 축소 → icon-48.png
   - 16x16으로 축소 → icon-16.png

### 온라인 도구 사용

**옵션 1: Canva**
1. 사용자 정의 크기 생성: 128x128px
2. "루프 화살표" + "음표" 검색
3. 요소 결합
4. PNG로 다운로드 (투명 배경)
5. 온라인 도구로 다른 크기 조정

**옵션 2: Figma 커뮤니티**
1. "음악 아이콘" 또는 "루프 아이콘" 검색
2. 프로젝트에 복제
3. 색상 커스터마이징
4. 모든 크기 내보내기

## ✅ 아이콘 체크리스트

### 디자인 품질
- [ ] 16px에서 128px까지 잘 축소됨
- [ ] 썸네일 크기에서 인식 가능
- [ ] 밝은 배경과 어두운 배경 모두에서 보임
- [ ] 작아지면 사라지는 세밀한 디테일 없음
- [ ] 전문적인 외관
- [ ] 독특하고 기억에 남음

### 기술 요구사항
- [ ] 투명 배경 PNG 형식
- [ ] 정확한 크기: 16x16, 48x48, 128x128
- [ ] RGB 색상 모드
- [ ] 각 파일 크기 1MB 미만 (일반적으로 <100KB)
- [ ] 저작권 위반 없음
- [ ] 모든 크기에서 일관된 스타일

### 브랜드 정렬
- [ ] 음악/연습 테마 표현
- [ ] 루프/반복 컨셉 포함
- [ ] 전문적이고 신뢰할 수 있음
- [ ] 타겟 청중(음악가)에게 매력적

## 🚫 피해야 할 일반적인 실수

1. **너무 많은 디테일**: 가는 선은 16px에서 사라짐
2. **낮은 대비**: 다른 배경에서 보기 어려움
3. **일관되지 않은 스타일**: 다른 크기에서 다른 모습
4. **저작권이 있는 심볼**: 브랜드 로고를 정확히 복사하지 마세요
5. **작은 아이콘의 텍스트**: 16px 또는 48px에서 읽을 수 없음
6. **패딩 없음**: 가장자리에 닿는 아이콘은 답답해 보임
7. **잘못된 형식**: PNG 대신 JPEG (투명도 없음)
