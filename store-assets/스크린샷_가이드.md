# 스크린샷 가이드

## 📸 필수 스크린샷

**형식**: PNG 또는 JPEG
**크기**: 1280x800 픽셀 (16:10 비율) - **권장**
**최소**: 640x400 픽셀
**최대**: 5개 스크린샷
**파일 크기**: 각 5MB 미만

---

## 📋 스크린샷 목록 (우선순위 순)

### 1. 메인 인터페이스 - 여러 연습 섹션 ⭐⭐⭐
**파일명**: `01-main-interface.png`
**캡션**: "여러 연습 섹션을 루프 카드로 정리하세요"

**보여줄 내용**:
- 3-4개의 루프 섹션 카드가 보임
- 다른 라벨들: "Intro", "Verse", "Chorus", "Solo"
- 펼쳐진 카드와 접힌 카드 혼합
- 시간 범위가 명확하게 보임 (예: "0:15 - 0:30")
- 재생 속도 표시 (예: "0.75x", "1.0x")
- 활성 루프 강조 표시 (파란색/색상 테두리)
- 배경에 YouTube 비디오 재생 중

**모범 사례**:
- 인기있는 음악 튜토리얼 비디오 사용 (기타, 피아노, 드럼)
- 루프 길이의 다양성 보여주기
- 최소 하나의 접힌 카드 포함
- 텍스트가 읽기 쉬운지 확인

---

### 2. 새 루프 생성 ⭐⭐⭐
**파일명**: `02-create-loop.png`
**캡션**: "마디 기반 길이 또는 사용자 정의 시간으로 몇 초 만에 루프 생성"

**보여줄 내용**:
- "새 루프" 생성 인터페이스가 보임
- 마디 길이 버튼 강조 (2, 4, 8, 16 마디)
- 라벨 프리셋 드롭다운 열림:
  - Intro
  - Verse
  - Chorus
  - Bridge
  - Outro
  - Custom...
- 현재 비디오 타임스탬프가 보임
- Tempo (BPM) 및 Time Signature 필드 채워짐

**스테이징**:
```
현재 시간: 1:23
길이: [2] [4] [8] [16] 마디
라벨: [Chorus ▼]
Tempo: 120 BPM
Time Sig: 4/4
```

### 3. 템포 설정 & TAP 기능 ⭐⭐
**파일명**: `03-tempo-control.png`
**캡션**: "TAP 기능으로 템포를 설정하고 박자 표시 선택"

**보여줄 내용**:
- 템포 섹션 펼쳐짐
- BPM 필드: "124"
- TAP 버튼 두드러짐
- 박자 표시 드롭다운에 옵션 표시 (2/4, 3/4, 4/4 등)
- TAP 버튼 위에 커서가 있을 수도 있음

**컨텍스트**:
명확한 템포가 있는 노래로 보여주기 (록, 팝, 재즈)
전문적인 음악 연습 설정 시연

---

### 4. 섹션별 속도 조절 ⭐⭐
**파일명**: `04-speed-control.png`
**캡션**: "각 섹션마다 독립적으로 재생 속도 조정"

**보여줄 내용**:
- 다른 속도의 동일한 루프 섹션:
  - 루프 1 "Intro": 1.0x
  - 루프 2 "Verse": 0.75x (강조 - 연습 중)
  - 루프 3 "Chorus": 0.5x
  - 루프 4 "Solo": 0.25x
- 속도 조절 슬라이더 또는 버튼이 보임
- 점진적 연습 시연 (느림 → 빠름)

---

### 5. 정리 기능 ⭐
**파일명**: `05-organization.png`
**캡션**: "드래그로 재정렬, 접기로 집중, 마스터하면 삭제"

**보여줄 내용**:
- 카드에 드래그 핸들이 보임 (드래그 중)
- 접힌 카드와 펼쳐진 카드 혼합
- "..." 메뉴가 열려 옵션이 보일 수도:
  - 삭제
  - 시작 시간 설정
  - 종료 시간 설정
- 깔끔하고 정리된 작업 공간

## 🎨 스크린샷 준비 체크리스트

### 촬영 전

- [ ] 실제 YouTube 비디오 사용 (음악 튜토리얼 권장)
- [ ] 명확한 섹션이 있는 비디오 선택 (인트로, 벌스, 코러스 등)
- [ ] 4-5개의 현실적인 연습 루프 설정
- [ ] 모든 필드 채우기 (라벨, 템포, 박자 표시)
- [ ] 라이트 모드와 다크 모드 모두 테스트
- [ ] 브라우저 알림/확장 프로그램 지우기

### 촬영 중

- [ ] 전체 화면 브라우저 (F11) 또는 최대화된 창
- [ ] 브라우저 북마크 바 숨기기
- [ ] 가능하면 YouTube 상단 네비게이션 숨기기
- [ ] 100% 줌 (Ctrl+0)
- [ ] 정확히 1280x800으로 캡처 또는 비례적으로 축소
- [ ] 텍스트가 선명하고 읽기 쉬운지 확인

### 촬영 후

- [ ] 정확히 1280x800으로 자르기
- [ ] 파일 크기 최적화 (PNG 압축 사용)
- [ ] 필요시 미묘한 드롭 섀도우 추가
- [ ] 썸네일 크기에서 텍스트 가독성 확인
- [ ] 파일 번호 매기기: 01-, 02-, 03-, 등

## 🎬 권장 비디오 예시

스크린샷에 사용하기 좋은 비디오:

1. **기타 레슨**
   - "쉬운 기타 리프 튜토리얼"
   - 명확한 섹션: 인트로, 메인 리프, 아웃트로
   - 2-3분 길이

2. **피아노 튜토리얼**
   - "피아노 코드 진행"
   - 벌스, 코러스 구조
   - 인기 곡 커버

3. **드럼 레슨**
   - "초보자 드럼 비트"
   - 명확한 템포
   - 각 드럼 패턴별 별도 섹션

4. **베이스 튜토리얼**
   - "펑크 베이스 라인 레슨"
   - 그루비하고 리드미컬
   - 여러 섹션

## 📝 캡션 작성 팁

캡션은:
- **짧게**: 1줄, 100자 미만
- **설명적**: 스크린샷이 무엇을 보여주는지 설명
- **이점 중심**: 사용자 가치 강조
- **행동 지향적**: 동사 사용 (생성, 조정, 정리)

**좋은 예시**:
✅ "마디 기반 길이로 몇 초 만에 루프 생성"
✅ "각 섹션마다 독립적으로 재생 속도 조정"
✅ "드래그 앤 드롭으로 연습 섹션 정리"

**나쁜 예시**:
❌ "앱의 스크린샷"
❌ "이것이 인터페이스입니다"
❌ "Loop Practice for YouTube"

## 🎯 촬영 팁

### 실제 사용 시나리오 만들기
1. YouTube에서 인기 기타 튜토리얼 열기
2. 3-4개의 현실적인 루프 생성:
   - "Intro" (0:00-0:08, 1.0x)
   - "Verse" (0:15-0:30, 0.75x)
   - "Chorus" (0:45-1:00, 0.5x)
   - "Solo" (1:30-1:45, 0.25x)
3. Tempo를 120 BPM으로 설정
4. Time Signature를 4/4로 설정
5. 하나의 루프를 활성화 (파란 테두리)
6. 스크린샷 촬영

### 조명과 대비
- 밝은 테마 사용 (더 나은 가독성)
- 또는 다크 테마 (YouTube와 매칭)
- 텍스트가 배경과 대비되는지 확인
- 그림자나 눈부심 없음

### 구도
- 확장 프로그램 UI를 중앙에 배치
- YouTube 비디오가 컨텍스트로 보임
- 너무 많은 빈 공간 없음
- 중요한 버튼/텍스트가 잘림 없음
