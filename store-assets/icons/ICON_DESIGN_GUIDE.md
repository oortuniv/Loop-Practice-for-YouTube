## ğŸ¯ Design Specifications

### 16x16 Icon (Toolbar)
- **Keep it simple**: Maximum 2 colors
- **High contrast**: Must be visible on light and dark backgrounds
- **Minimal detail**: Avoid fine lines
- **Center the symbol**: Leave 2px padding on all sides
- **Test visibility**: Check against different browser themes

**Recommended**:
```
Simple circular arrow (âŸ³) in YouTube blue
OR
Musical note (â™ª) in black
```

### 48x48 Icon (Management Page)
- **More detail allowed**: Can include 2-3 elements
- **Clear at thumbnail size**: Still recognizable when small
- **Balanced composition**: Center-weighted design
- **Leave padding**: 6-8px margin

**Recommended**:
```
Circular arrow (âŸ³) + musical note (â™ª)
OR
Guitar silhouette + loop symbol
```

### 128x128 Icon (Store Listing)
- **Full detail**: Can include text, gradients, effects
- **Brand identity**: Most visible version
- **Professional quality**: This is your "storefront"
- **Leave padding**: 16-20px margin

**Recommended**:
```
Gradient background (blue to light blue)
Large circular arrow with musical note
Optional: "LP" text or instrument silhouette
Subtle drop shadow for depth
```

## ğŸš« Common Mistakes to Avoid

1. **Too much detail**: Fine lines disappear at 16px
2. **Low contrast**: Hard to see on different backgrounds
3. **Inconsistent style**: Different looks at different sizes
4. **Copyrighted symbols**: Don't copy brand logos exactly
5. **Text in small icons**: Unreadable at 16px or 48px
6. **No padding**: Icon touching edges looks cramped
7. **Wrong format**: JPEG instead of PNG (no transparency)
